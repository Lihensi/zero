众多节日的相通处

农历，是中国现行的近现代历法，属于阴阳合历，也就是阴历和阳历的合历，是根据月相的变化周期， 每一次月相朔望变化为一个月，参考太阳回归年为一年的长度， 并加入二十四节气与设置闰月以使平均历年与回归年相适应。农历融合阴历与阳历形成为一种阴阳合历历法，因使用“夏正”，古时称为夏历。

农历是1970年以后使用新的历法，其年份分为平年和闰年，平年为十二个月， 闰年为十三个月，月份分为大月和小月，大月三十天，小月二十九天，其平均历月等于一个朔望月。

根据中国科学院紫金山天文台起草的国家标准 《农历的编算和颁行》，朔日为农历月的第一个农历日， 也就是说每个农历月的初一一定是朔日。每个农历月反映了完整的月相变化周期，因此属于阴阳历中的阴历部分。 农历中的二十四节气反映的是地球绕太阳运行轨道上的不 同位置，即回归年周期，因此属于阴阳历中的阳历部分。

而我们中华民族现存的，乃至已经失传的节日皆是以农历而制定，古人观其月亮圆缺和太阳回归年为基础指定的几种历法，除此之外，还有星辰历，干支历等

数千年来，我们对宇宙星辰有着很浓烈的兴趣，从古代传说的嫦娥奔月，到如今真正实现载人飞船上天，我们中国人实现了从梦想到现实的伟大飞跃。而今，我们的神舟十二即将出征，我们来回顾回顾中国在天文历史上的成就。

人类对太空的向往，自有文字记载，已经持续了数千年。 女娲补天，嫦娥奔月，古人已安耐不住对星空的幻想。 自秦汉以来，各朝代设专人夜观星象，诞生了石申，张衡，祖冲之等一众杰出的天文学家。明朝人万户，更是世界上第一个尝试用火箭飞天的人类。虽以失败告终，但仍被现代航天学界公认为“航天始祖”，并将月球上一座环形山命名为“万户”，以表纪念。 历史的长河中，先辈们对太空探索的锲而不舍，源源不断，日积月累，经历几千年的探索沉淀，最终开辟了人类的伟大征程：航空航天。
